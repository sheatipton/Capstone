# Fawn

## Description
Fawn is an innovative iOS application designed to connect users to nonprofits in their area in order to make donation processes easier, faster, and more effective for directly supporting those in need.

## Tools
xCode, SwiftUI, Google Firebase, various APIs by Apple, Figma, Canva
